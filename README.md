# 📊 HR Attendance & Workforce Analytics Dashboard

## 📌 Project Description

The **HR Attendance & Workforce Analytics Dashboard** is a Power BI-based data analytics project designed to track and analyze employee attendance, work-from-home (WFH) trends, and sick leave (SL) patterns.

This project helps HR teams gain meaningful insights into workforce behavior, enabling data-driven decisions to improve productivity, attendance policies, and employee management.

---

## 🚀 Key Features

- 📈 **Attendance Monitoring**
  - Tracks overall attendance percentage  
  - Visualizes attendance trends over time  

- 🏠 **Work From Home (WFH) Analysis**
  - Daily and trend-based WFH insights  
  - Helps understand remote work patterns  

- 🏥 **Sick Leave (SL) Tracking**
  - SL percentage analysis  
  - Identifies unusual leave trends  

- 👨‍💼 **Employee-Level Insights**
  - Individual attendance records  
  - Present days and WFH contribution  

- 📅 **Dynamic Filtering**
  - Month-wise filters (Apr, May, Jun)  
  - Custom date range selection  

- 📊 **Day-wise Analysis**
  - Attendance, WFH, and SL by weekday  
  - Helps optimize workforce planning  

---

## 🛠️ Tech Stack

- Power BI Desktop  
- Microsoft Excel (.xlsx)  
- DAX (Data Analysis Expressions)  
- Data Modeling
